# [BOJ 10171 Link](https://www.acmicpc.net/problem/10171)
#### 문제
아래 예제와 같이 고양이를 출력하시오.
```
\    /\
 )  ( ')
(  /  )
 \(__)|
```
#### 입력
없음.

#### 출력
고양이를 출력한다.
- - -
사용하는 언어로 문제에 주어진 고양이를 똑같이 출력하면 된다.

이스케이프 문자를 적절히 사용할 수 있는지 물어보는 문제같다.
### Python3
```
print("\\    /\\")
print(" )  ( ')")
print("(  /  )")
print(" \\(__)|")
```
### C/C++
```
#include <stdio.h>
int main(){
	printf("\\    /\\\n");
	printf(" )  ( \')\n");
	printf("(  /  )\n");
	printf(" \\(__)|");
	return 0;
}
```
